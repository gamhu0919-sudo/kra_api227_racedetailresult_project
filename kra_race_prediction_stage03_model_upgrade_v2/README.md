# Stage 03: Model Upgrade v2 (고도화 모델)

본 단계는 기존 Stage 03의 베이스라인 모델을 고도화하기 위한 프로젝트입니다. 출전두수 보정, 최근성, 거리/등급별 적성 등을 반영한 신규 피처를 생성하고, 이를 활용해 성능이 개선된 신규 모델(Model A, B)을 학습 및 평가합니다.

## 📂 폴더 구조
- `src/`: 데이터 로드, 피처 엔지니어링, 학습, 평가 스크립트
- `data/`: 로드된 데이터 및 신규 생성된 피처 데이터셋
- `models/`: 고도화된 모델 파일 (.pkl)
- `outputs/`: 성능 비교 테이블, 지표, 진단 결과
- `reports/`: 최종 분석 보고서

## 🚀 실행 방법
루트 디렉토리에서 아래 명령어를 실행하면 모든 단계가 순차적으로 처리됩니다.
```bash
uv run python run_pipeline.py
```

## 📈 주요 피처 고도화 내용
1. **출전두수 보정 순위**: 경주별 인원 차이를 반영한 상대 성적 지표.
2. **최근성 보정**: 최근 3, 5경주의 성과를 가중 반영.
3. **거리/등급 적성**: 특정 환경에서의 과거 성적 및 Fallback 로직 적용.
4. **휴식 기간**: 이전 경주 이후 경과일수 반영.

## 📊 성능 개선 결과
- **Precision@3**: 기존 47.95% → **51.60% (+3.65%p)**
- **Hit@3**: 기존 58.08% → **93.27% (+35.19%p)**
