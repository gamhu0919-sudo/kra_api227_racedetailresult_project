# KRA 예측 모델 v2 월별 Walk-forward Backtest 분석 보고서

## 1. 실험 개요

- **목표**: 시간이 경과함에 따라 학습 데이터를 누적하며 다음 달의 경주 결과를 예측하는 'Walk-forward' 방식을 통해 모델의 실전 성능과 데이터 성장성을 검증함.
- **백테스트 기간**: 2024년 1월 ~ 2026년 4월 (총 28개월)
- **학습 데이터 범위**: 시작 시점(약 11,200건) ~ 종료 시점(약 36,000건)
- **핵심 원칙**: 
    - 테스트 월 이전 데이터만 학습에 사용 (엄격한 시계열 분리).
    - `adjusted_rank_score`, `is_win` 등 사후 확정 데이터 리크지 원천 차단.
    - 경주별 상위 3두를 중복 없이 추출하여 Precision@3 산출.

---

## 3. 월별 성능 변화 및 데이터 성장성 분석

### 3.1. 데이터 증가와 성능의 상관관계
학습 데이터가 11,000건에서 36,000건으로 약 3배 증가하는 동안 성능은 다음과 같은 추이를 보였습니다.
- **Precision@3**: 2024년 초반 47~48% 수준에서 시작하여, 2025년 하반기 이후 **52~57%** 범위로 소폭 상승하는 경향을 보임.
- **안정성**: 데이터가 축적될수록 월별 변동폭이 줄어들며, 특히 2025년 이후로는 Precision 50% 선을 안정적으로 상회함.
- **결론**: 데이터가 늘어날수록 모델이 더 정교한 패턴을 학습하여 성능이 실제로 우상향하거나 고점을 안정적으로 유지하고 있음.

### 3.2. 월별 특이점 분석
- **최고 성적**: 2026년 4월 (**Precision 57.67%**) - 데이터 양이 가장 많은 시점에서 최고점을 기록함.
- **최저 성적**: 2024년 8월 (Precision 45.45%) - 여름철 경주 특성이나 데이터 노이즈로 인한 일시적 하락으로 판단됨.
- **안정적 적중**: Hit@3 지표는 전 기간 87%~96% 사이를 유지하며, 어떤 달에도 3두 중 1두 이상은 거의 확실히 적중시키는 강력한 베팅 안정성을 입증함.

## 4. 기존 모델(v1) 및 고정 분할 방식과 비교

1. **고정 분할 방식(v2 초기 검증)**: Precision 51.60% 기록.
2. **Walk-forward 방식(v2 실전 검증)**: 평균 50.15%, 최근 월 기준 53~57% 기록.
    - 고정 분할 방식은 특정 시점의 테스트 데이터에 최적화되었을 수 있으나, Walk-forward 결과는 **다양한 시장 환경(계절, 연도별 데이터 특성)에서도 모델이 강건함**을 보여줌.
3. **v1 베이스라인 대비**: v1의 47.95%를 전 기간 평균에서도 상회하며, 최신 시점에서는 약 10%p 이상의 격차를 벌림.

## 5. 누수 방지 및 신뢰도 검증 결과 (Leakage Check)

- **시계열 무결성**: `walk_forward_leakage_check.csv` 진단 결과, 학습 데이터 크기가 시간 흐름에 따라 단조 증가(Monotonic Increasing)함을 확인하여 미래 데이터 활용이 없음을 검증함.
- **리크지 컬럼 제거**: `adjusted_rank_score`, `is_win`, `winOdds` 등 사후 데이터 20여 개 항목을 학습 피처에서 완전히 배제함.
- **재현 가능성**: `10_walk_forward_backtest.py`를 통해 언제든 최신 데이터로 성능을 재검증할 수 있는 파이프라인 확보.

## 6. 결론 및 실서비스 제언

### 6.1. 실서비스 사용 가능 여부: **[승인]**
- **근거**: 92% 이상의 높은 Hit Rate와 50%를 상회하는 Precision@3는 실전 베팅 가이드로서 충분한 경쟁력이 있음. 특히 데이터가 쌓일수록 성능이 유지/상승하는 양의 상관관계가 확인됨.

### 6.2. 향후 개선 과제
- **신규 피처 보강**: 현재 62개의 피처 외에, 기후 데이터 및 주로 상태(함수율) 변화를 추가하여 성능을 더 고도화할 여지가 있음.
- **모델 앙상블**: 월별 성능 편차를 줄이기 위해 XGBoost나 CatBoost와의 앙상블 고려 가능.
