# KRA Top3 예측 모델 고도화 v2 보고서

## 1. 작업 목적
기존 베이스라인 모델의 한계(단순 평균순위 의존)를 극복하기 위해 출전두수 보정 순위, 최근성, 거리 및 등급 적성 피처를 도입하여 예측 모델을 고도화함.

## 2. 기존 모델 한계
- 약한 등급에서의 좋은 순위와 강한 등급에서의 평범한 순위를 구분하지 못함.
- 출전두수에 따른 순위의 가치를 반영하지 못함.
- 오래전 성적과 최근 성적을 동일 비중으로 처리함.

## 3. 새로 만든 피처
- **출전두수 보정 순위 (`adjusted_rank_score`)**: `1 - ((rank - 1) / (field_size - 1))` 공식을 통해 출전두수가 달라도 비교 가능한 상대적 성과 지표 산출.
- **최근성 보정 성적**: 최근 3, 5경주의 평균 보정 순위 및 Top3율 산출.
- **거리 및 등급 적성**: 특정 거리 구간(단거리/마일/장거리) 및 등급에서의 과거 성과를 반영하고, 데이터 희소성 해결을 위한 Fallback 로직 적용.
- **휴식 기간 및 유형**: 이전 경주로부터의 경과일수 및 휴식 유형(short/normal/long) 반영.
- **경주 내 상대 피처 v2**: 위 신규 지표들을 경주 내에서 다시 순위화하여 상대적 우위 파악.

## 4. 데이터 누수 방지 방식
- 모든 과거 성적 기반 피처 생성 시 `groupby().shift(1)`를 적용하여 현재 경주의 결과가 피처에 포함되는 것을 엄격히 차단함.
- `leakage_check_v2.csv` 진단 결과, 모든 피처의 타겟 상관계수가 0.3 미만으로 정상 범주임을 확인함.

## 5. Model A / Model B 구성
- **Model A (v2_full)**: 기존 유효 피처 + 모든 신규 고도화 피처.
- **Model B (v2_adj_only)**: 기존 단순 평균순위 관련 피처를 제거하고 보정 순위 지표 중심으로 구성.

## 6. 기존 모델 대비 성능 비교 (Test Set 기준)

| 지표 | Baseline (Stage 03) | Model A (v2_full) | Model B (v2_adj_only) | 개선폭 (Model A vs Base) |
| :--- | :---: | :---: | :---: | :---: |
| **Precision@3** | 47.95% | **51.60%** | 51.15% | **+3.65%p** |
| **Hit@3** | 58.08% | **93.27%** | 92.69% | **+35.19%p** |
| **Avg Correct** | 1.438 | **1.548** | 1.535 | **+0.110** |

- Model A가 모든 지표에서 기존 모델 및 Model B를 상회함.
- 특히 Hit@3(상위 3두 중 1두 이상 적중)가 비약적으로 향상되어 모델의 예측 안정성이 크게 개선됨.
