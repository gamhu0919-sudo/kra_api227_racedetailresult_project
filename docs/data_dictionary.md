# 데이터 딕셔너리 (2025.03 - 2026.03)

## 1. 데이터셋 개요
- **기준**: 한국마사회 API227 경주별상세성적표
- **단위**: 경주(rc_id) + 출전마(chulNo) 개별 Row
- **위치**: `data/processed/race_detail_result_202503_202603_processed.csv`

## 2. 컬럼 상세 명세 (가공 및 파생 포함)

| 컬럼명 | 한글 설명 | 데이터 타입 | 예시값 | 특징 및 분석 활용 |
|--------|-----------|-------------|--------|-------------------|
| `rcDate` | 경주일자 | str (object) | `20250301` | 시간축 분석 (년/월/일) |
| `rcDate_dt` | 경주일자(DT) | datetime | `2025-03-01` | 날짜 연산 및 시계열 분석 |
| `rcNo` | 경주번호 | float/int | `1` | 특정 일별 경주 순서 |
| `stOrd` | 착순 | float (int) | `1.0` | **Target Variable** (순위) |
| `chulNo` | 출전번호 | str/int | `6` | 마번 |
| `hrNo` | 마필번호 | str | `033211` | 말 식별자 (과거 전적 추적용) |
| `hrName` | 마필명 | str | `금아백` | 데이터 식별 보조 |
| `meet` | 경마장코드 | int/str | `1` | 지역 코드 (1:서울, 2:제주, 3:부경) |
| `meet_nm` | 경마장명 | str | `서울` | 가독성용 파생 컬럼 |
| `sex` | 성별 | str | `수`, `암`, `거` | 성별 퍼포먼스 차이 분석 |
| `age` | 마령 | int | `3` | 나이에 따른 숙련도/노쇠화 분석 |
| `wgBudam` | 부담중량 | float | `53.0` | 성적과의 상관관계 핵심 변수 |
| `jkName` | 기수명 | str | `조상범` | 기수 숙련도 및 조합 분석 |
| `trName` | 조교사명 | str | `박병일` | 조교사(훈련) 성과 분석 |
| `owName` | 마주명 | str | `최강자` | 마주별 투자 성향 분석 |
| `win` | 단승배당 | float | `2.4` | 시장 인기 지표 |
| `plc` | 연승배당 | float | `1.2` | 안전성 지표 |
| `rcTime` | 경주기록 | float | `71.2` | 속도 분석 (초 단위) |
| `hrRating` | 레이팅 | float | `45.0` | 마필 공식 능력치 |
| `horse_weight` | 현재체중(kg) | float | `480.0` | 체격 조건과 주력의 관계 |
| `horse_weight_delta` | 체중증감(kg) | float | `-2.0` | 컨디션 지표 (급격한 증감 주의) |
| `is_winner` | 우승 여부 | bool | `True` | 1착 여부 (Binary Label) |
| `is_place` | 입상 여부 | bool | `True` | 3착 이내 여부 (Binary Label) |

## 3. 코드값 관리
- `meet`: 1(서울), 2(제주), 3(부경)
- `sex`: 수(수말), 암(암말), 거(거세마)
- `chulYn`: 1(출전), 0(미출전) - 기본적으로 1인 데이터만 전처리 대상 권장
