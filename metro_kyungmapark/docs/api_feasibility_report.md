# 경마공원역 승하차 인원 데이터 수집 및 API 타당성 검토 보고서

## 1. 개요
본 보고서는 서울교통공사 API를 활용하여 4호선 경마공원역의 승하차 인원 데이터를 수집하려는 시도와 그 기술적 검증 결과를 기록한 문서입니다.

## 2. 사용 API 상세
- **API 명칭**: 서울교통공사_역별승하차인원
- **엔드포인트**: `https://apis.data.go.kr/B553766/psgr/getStnPsgr`
- **조회 기간**: 2026-04-12 ~ 2026-04-18 (최근 7일)
- **요청 파라미터**:
    - `stnNm`: 경마공원, 경마공원역
    - `lineNm`: 4호선, (빈값)
    - `dataType`: JSON

## 3. 검증 결과
### [결론: 수집 불가]
조회 시도 기간 전체에 대해 경마공원역의 데이터가 응답되지 않았습니다 (`totalCount: 0`).

### 실패 원인 분석
1. **운영 주체 불일치**: 4호선 경마공원역은 **한국철도공사(KORAIL)**가 운영하는 과천선 구간에 위치합니다. 서울교통공사 API는 자사 관할 역(1~8호선 일부 및 서울시 구간)의 데이터만 제공하므로 경마공원역 데이터가 누락된 것으로 확인되었습니다.
2. **데이터 제공 범위 제한**: 해당 API는 기술 명세상 서울교통공사 관할 외의 역 데이터는 제공하지 않음을 실제 호출을 통해 증명하였습니다.

## 4. 1년치 데이터 확보 가능 여부
- **불가능**: 서울교통공사 API(`getStnPsgr`)로는 경마공원역의 과거 1년치 데이터를 확보할 수 없습니다.

## 5. 대안 데이터 소스 제안
경마공원역 데이터가 반드시 필요한 경우 아래의 공식 소스를 검토해야 합니다.

1. **공공데이터포털(data.go.kr)**:
    - `한국철도공사_광역철도 역별 승하차 인원 현황` 오픈 API
    - `한국철도공사_광역철도 수송인원 및 승하차인원` 파일 데이터 (CSV)
2. **서울 열린데이터 광장(data.seoul.go.kr)**:
    - `서울시 지하철 호선별 역별 승하차 인원 정보` (서울 시내 역 중심이지만 코레일 구간 포함 여부 확인 필요)
3. **KORAIL 철도통계**:
    - 한국철도공사에서 매년 발행하는 철도통계연보 활용

## 6. 작업 결과물

- **수집 데이터**: 없음 (빈 파일 생성 지양)
